### WeChatBackupNoRoot 

免ROOT微信聊天数据备份/导出工具



该版本软件特点:

- 操作简单,无需额外软件配合
- 无需手机ROOT
- 无需电脑配合
- 无需解密,数据明文保存

开箱即用: [工具下载页面](https://github.com/MiDuoKi/WechatBackupNoRoot/releases)
