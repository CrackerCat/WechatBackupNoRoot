### WeChatBackupNoRoot 

免ROOT微信聊天数据备份/导出工具


#### 使用场景描述:

如果你不得不删除你的微信聊天记录，但是想保存某人的对话记录。可以使用本软件进行聊天备份。

相对于其他备份方案，在使用上不需要任何其他工具配合。也不用登录电脑客户端去做备份。也不用进行

聊天文件的加解密操作。旨在一个简单实用。当然他并不完美，但好在简单易用。


#### 该版本软件特点:

- 操作简单,无需额外软件配合
- 无需手机ROOT
- 无需电脑配合
- 无需解密,数据明文保存

开箱即用: [工具下载页面](https://github.com/MiDuoKi/WechatBackupNoRoot/releases)
